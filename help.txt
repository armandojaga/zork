Welcome to Zork help!

The format is: command (shorcut) [target]
Available commands:
[direction] : travel to the specified direction
	north(n), south(s), east(e), west(w), northeast(ne), northwest(nw), southeast(se), southwest(sw), up(u), down(d)
go [direction] : travel to the specified direction
attack (kill) [enemy] : attacks a particular enemy
examine [item] : shows the item's description
take [item|all] : add an item to the player's inventory, use "all" to add all available items in the scene
drop [item|all] : remove a particular item from the player's inventory, user "all" to remove all items
inventory (i) : shows the items in the player's inventory
open [item] : opens an item that contains other items
look : looks around at current scene
help : displays this help
